# 极简记账本 (Simple Accounting)

一个基于HTML+JavaScript的轻量级记账应用，支持本地数据存储和Excel导出功能。

🔗 **在线体验**: https://mxiaobai-gyf.github.io/Track-expenses-easily/

---

## 功能特性

- ✅ 收入/支出记录（金额、分类、备注、时间）
- ✅ 实时余额计算与统计
- ✅ 本地数据持久化（LocalStorage）
- ✅ 数据导出为Excel格式
- ✅ 响应式设计，支持移动端访问
- ✅ 数据本地存储，隐私安全

---

## 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 | 页面结构 |
| CSS3 | 样式布局 |
| JavaScript | 业务逻辑与数据操作 |
| LocalStorage API | 本地数据存储 |
| SheetJS (xlsx.js) | Excel文件导出 |
